# 系统问答与操作手册管理平台 - 部署启动手册

## 项目结构

```
D:\QA\
├── front_service/           # Vue 3 前端项目
├── QA_service/              # Spring Boot 后端项目
└── init.sql                 # 数据库初始化脚本
```

---

## 一、数据库初始化

### 1. 启动 MySQL

确保 MySQL 服务已启动。

### 2. 执行初始化脚本

在 MySQL 中执行 `init.sql`：

```bash
mysql -uroot -pbomc0705 < D:\QA\QA_service\init.sql
```

或通过 MySQL 客户端工具（如 Navicat）导入 `init.sql` 文件。

---

## 二、后端启动（IDEA）

### 方法1：直接运行

1. 在 IDEA 中打开项目 `D:\QA`
2. 找到 `QA_service/src/main/java/com/qa/QaApplication.java`
3. 右键 → **Run 'QaApplication'**

### 方法2：Maven 运行

1. 在右侧 **Maven** 面板
2. 展开 **QA-service** → **Lifecycle**
3. 双击 **spring-boot:run**

### 验证

启动成功后访问：`http://localhost:8080`

---

## 三、前端启动（IDEA）

### 方法1：命令行

```bash
cd D:\QA\front_service
npm install
npm run dev
```

### 方法2：IDEA 配置

1. **Run** → **Edit Configurations**
2. 点击 **+** → **npm**
3. 配置：
   - Name: `front_service dev`
   - Package manager: `npm`
   - Command: `dev`
   - Working directory: `$PROJECT_DIR$/front_service`
4. 点击 **OK**
5. 运行配置

### 验证

启动成功后访问：`http://localhost:5173`

## 六、常见问题

### 1. 后端启动失败

- 检查 MySQL 是否启动
- 检查 `application.yml` 数据库密码是否正确

### 2. 前端无法访问后端 API

- 检查后端是否启动在 8080 端口
- 检查前端 `vite.config.ts` 代理配置

### 3. 编译错误

- 确保 JDK 版本为 21
- 在 IDEA 中刷新 Maven 依赖
